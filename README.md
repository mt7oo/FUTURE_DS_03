## Project Overview
This project analyzes marketing funnel performance and customer conversion behavior using Power BI. The dashboard helps identify conversion rates, customer drop-off stages, and campaign effectiveness using the Bank Marketing Dataset.

## Objectives
- Analyze lead-to-customer conversion
- Identify major funnel drop-offs
- Compare conversion performance across customer groups
- Generate business recommendations


## Dashboard Features
- KPI Cards
- Funnel Chart
- Conversion Analysis
- Interactive Slicers
- Business Insights & Recommendations

## Key Insights
- High drop-off before final conversion
- Cellular contact performs best
- Some job categories convert more
- Too many contacts reduce conversions

## Recommendations
- Focus on high-converting channels
- Improve lead follow-up strategy
- Reduce excessive customer contacts
- Target better-performing customer groups
